Help & Information

Welcome to the Ministry of Justice File Tracking System. This platform helps you manage, track, and transfer files securely and efficiently within the ministry.

  • Use the Dashboard to see stats and recent activity.
  • Go to Files to view, search, and manage all files.
  • Use Transfers to move files between departments and locations.
  • Click your profile icon for settings and to sign out.
  • Need to request a file? Use the Request File button on the dashboard.
  • For further assistance, contact the system administrator.
